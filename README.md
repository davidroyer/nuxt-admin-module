## Setup
- Add `nuxt-admin-module` dependency using yarn or npm to your project
- Add `nuxt-admin-module` to `modules` section of `nuxt.config.js`

```js
{
  modules: [
    // Simple usage
    'nuxt-admin-module',

    // With options
    ['nuxt-admin-module', { /* module options */ }],
 ]
}
```
